# TL-Intl E-commerce Website

## Project Description
TL-Intl is a comprehensive e-commerce platform designed to provide a seamless online shopping experience for health-conscious customers looking for international supplement brands. This web application focuses on offering a wide variety of health supplements from recognized brands across the globe.

## Features
- **User Authentication**: Secure sign-up and login functionality.
- **Product Catalog**: Browse through a diverse range of health supplements with filters and search capabilities.
- **Shopping Cart**: Add products to a cart for convenient purchasing.
- **Secure Payment Options**: Multiple payment methods including credit/debit cards, PayPal, etc.
- **Order Tracking**: Track your orders from placement to delivery.
- **User Reviews**: Customers can leave reviews and ratings for purchased products.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Getting Started
To get started with the TL-Intl e-commerce platform, follow these steps:

### Prerequisites
- Node.js (v14 or newer)
- npm (v5 or newer)
- MongoDB (for database)

### Installation Instructions
1. **Clone the repository**
   ```bash
   git clone https://github.com/intlglobalfoods-dot/tl-intl.git
   cd tl-intl
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   Create a `.env` file in the root directory and add the necessary configuration for your database and other services.

4. **Run the application**
   ```bash
   npm start
   ```
   The application should now be running at `http://localhost:3000`.

5. **Access the UI**
   Open your web browser and navigate to the application URL.
